# Super Elad CV Game

A playable CV platformer game built with TypeScript and HTML5 Canvas.

## Features
-   **Custom Game Engine**: Built from scratch to demonstrate understanding of game loops, physics, and rendering.
-   **Interactive CV**: Explore my skills and experience by playing the game.
-   **Dockerized**: Easy to deploy and run anywhere.

## Quick Start

### Using Docker
```bash
docker-compose up --build
```
Visit `http://localhost:8080`

### Local Dev
```bash
npm install
npm run dev
```
Visit `http://localhost:3000`

## Controls
-   **Move**: Arrow Keys / WASD
-   **Jump**: Space / Up
